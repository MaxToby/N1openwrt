# 固件说明
* 本项目下编辑生成的固件是N1固件，因需要F大内核支持，故此为半成品，小白请勿下载，或到这里下载成品[N1固件](https://github.com/wxfyes/N1packaging/releases)。
* 如果你有编译打包基础，可以下载此页面固件配置F大内核进行打包成品！
